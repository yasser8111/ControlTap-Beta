# CustomizeNT (Beta)

![Zen Minimalist Preview 1](previwe-1.png)
![Zen Minimalist Preview 2](previwe-2.png)

CustomizeNT is a professional, minimalist, and highly customizable Chrome/Edge "New Tab" extension. It provides a zen-like experience for managing your favorite links, equipped with dynamic pages, widgets, and an elegant design tailored for maximum productivity.

## ✨ Key Features

- **Multi-Page Navigation**: Organize your links across multiple, easily manageable workspace pages with smooth transitioning.
- **Smart Widgets**: Includes a dynamic clock widget supporting both Digital (Rectangular) and Analog (Square) minimalist displays.
- **Advanced Customization**:
  - Adjust column counts (2-12) and scale cards (60% - 150%) to fit any screen resolution.
  - Built-in elegant search bar with adjustable visibility and scaling.
  - Save your personalized designs as **Custom Templates** and easily manage or delete them.
- **Fluid UI/UX Interactions**: 
  - **In-place Editing**: Seamless, prompt-free editing for renaming groups and pages.
  - Hover-based interactions to maintain a clean, distraction-free environment.
- **Interface Modes**: 
  - **Standard**: Full titles and contextual guides.
  - **Simple Mode**: Icon-only navigation for the ultimate minimalist setup.
- **Immersive Backgrounds**: Comes with 11 hand-picked premium video/image templates, plus support for uploading your own high-res media directly to local internal storage.
- **True Multilingual Support**: Full built-in localization mechanism with English (LTR) and Arabic (RTL) seamlessly integrated.

## 🚀 Getting Started

1. Download or clone this repository.
2. Open Chrome/Edge and navigate to `chrome://extensions` or `edge://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the project folder.

## 🛠️ Technology Stack

- **Pure HTML5/CSS3**: No heavy frameworks, ensuring instantaneous load times.
- **Vanilla JavaScript**: Optimized performance structured around a modular MVC architecture.
- **IndexedDB**: Persistent and robust local storage for user-uploaded custom media.
- **SortableJS**: Smooth, native-feeling drag and drop system.
- **Lucide Icons**: Clean, consistent, and beautiful iconography.

## 🎨 Interface Controls

- **Pages Tabs**: Create, rename, or switch between distinct workspace pages.
- **Templates Library**: Choose from beautiful curated backgrounds or create your own custom presets.
- **Palette**: Live-tweak primary accent colors, UI card opacity, and global themes (Light/Dark).
- **Settings Menu**: Granular control over grid layout, interface language, search engine display, and tab behaviors.
